Python —— *的用途

作者: umi
发表于2026-05-03
更新于2026-05-03
算法

* 在 Python 中有多个用途

1. 函数调用时——解包（Unpacking）

把一个列表、元组等可迭代对象“打散”成一个个独立的参数。

print 本来就可以接收多个参数，并用空格分隔输出。用 * 正好把列表拆成多个参数。

2. 函数定义时——接收任意多个位置参数（*args）

在形参前加 *，可以把调用时多余的位置参数打包成一个元组。

3. 赋值时——解包剩余元素

在变量前加 * 可以收集“剩下的”元素为一个列表。

4. 强制关键字参数（Python 3）

函数定义中，一个单独的 * 表示后面的参数必须用关键字传递。

5. 数学运算符——乘法和幂

这个最基础，但还是列出来方便对比。

  • 乘法：3 * 4 → 12
  • 幂（两个 *）：2 ** 3 → 8

总结对比

用法 语法 作用
函数调用解包 print(*list) 把列表拆成多个参数
函数定义收集参数 def f(*args) 多个参数打包成元组
赋值时收集剩余值 a, *b = ... 剩余元素收集成列表
强制关键字参数 def f(name, *, key) * 后的参数必须用关键字
乘法 a * b 算术乘法
幂 a ** b 乘方
